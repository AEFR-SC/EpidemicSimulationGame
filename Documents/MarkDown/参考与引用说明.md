**创新之处：将整个类需要调用的函数封装成了一个`run()`函数**， 例如：

```Python
class OneClass:  #别在意类名
    def __init__(self):
        pass
    def func1(self):
        pass
    def func2(self):
        pass
    def wrap(self):  #外部直接调用wrap即可
        self.func1()
        self.func2()
```
